# My Analytics App (Vite + React)

This is a starter, modular React project that includes:
- Vite + React
- TailwindCSS (config included)
- Framer Motion for animations
- react-dropzone for CSV upload
- Chart.js (via react-chartjs-2)

How to run:
1. `npm install`
2. `npm run dev`
3. Open the local dev server (Vite will show the URL)

Files of interest:
- `src/components` - UI components (Hero, Header, Tabs, Dropzone)
- `src/routes` - Pages: DataInput, Cleaning, Analytics, Insights, Predictions
- `src/utils` - CSV parser & sample data generator

Notes:
- This is a full project skeleton with working UI and basic logic ported from your HTML.
- The cleaning & prediction logic are simplified; replace with your original algorithms as needed.
